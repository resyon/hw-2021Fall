# hw-linux

## precondition
- cmake
- make
- g++ (support c++20)
- test on ubuntu21.10

```bash
# make install will install dependency from github
make install && make
```

## run
```bash
# server
./hw-2021Fall/build/hw-exe

# client
./hw-2021Fall-client/build/hw-2021Fall-client-exe
```

## clean
```bash
make clean
```
